# StreamLabs Chatbot Scripts

Here are a few playground scripts for making your Stream chat a bit more interactive.


## General

In oder to use the scripts in general, download StreamLabs Chatbot:
https://streamlabs.com/chatbot
Then make sure to connect yourself as well as a bot with your Stream.
If not done already install Python as explained on the Streamlabs Chatbot website.

Furthermore, you want to try scripting a neat little tool for yourself, here is a good API documentation for the stream chat:
https://github.com/AnkhHeart/Streamlabs-Chatbot-Python-Boilerplate/wiki/

For more information and usages of each script please refer to the README of said script.
